Atualmente trabalho com **Python, SQL, ETL, APIs, manipulação de planilhas, extração de dados e automação de rotinas corporativas**, criando soluções que transformam processos manuais em fluxos mais rápidos, organizados e rastreáveis.

Tenho maior familiaridade com a área de **dados**, mas também desenvolvo projetos voltados para **programação, automação, integração de sistemas e processamento de informações**.
